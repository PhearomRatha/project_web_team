ស្លាបមាន់បំពងទឹកអំពិលទុំ

គ្រឿងផ្សំ

- ស្លាបមាន់

- ទឹកអំពិលទុំ

- ខ្ទឹមស ចិញ្ច្រាំ

- ម្សៅស៊ុប ស្ករត្នោត ទឹកប្រេងខ្ចង

- ម្សៅឆា ( លាយជាមួយទឹកបន្តិច កូរអោយសព្វ )

- ម្សៅឆា ( លាយជាមួយទឹកបន្តិច កូរអោយសព្វ )

- ម្ទេស ហាន់តូច ( តិចរឺច្រើនតាមការចូលចិត្ត )

វិធីធ្វើ:

- យកស្លាបមាន់មកប្រលាក់ជាមួយម្សៅស៊ុបបន្តិច

- ដាក់ប្រេងឆាចូលក្នុងខ្ទះទុកអោយក្ដៅ រួចដាក់ស្លាបមាន់ចូល បំពងអោយឆ្អិន រួចដួសទុកមួយអន្លើ

- ដាក់ប្រេងឆាចូលក្នុងខ្ទះសារជាថ្មី ទុកអោយក្ដៅ រួចដាក់ខ្ទឹមសចូលបំពងអោយឈ្ងុយបន្តិច បន្ទាប់មកដាក់ស្ករត្នោត ទឹកអំពិលទុំ ម្សៅស៊ុប ទឹកប្រេងខ្ចង និង បន្ថែមទឹកបន្តិច ហើយកូរអោយសព្វចូលជាតិ រួចភ្លក់មើលអោយល្មមតានការចូលចិត្ត (រស់ជាតិជូរផ្អែម)

- រួចដាក់គល់ខ្ទឹម ម្ទេស និង ម្សៅឆាចូល កូរអោយសព្វ ទើបដាក់ស្លាប់មាន់ចូល រួចកូរត្រលប់ចុះត្រលប់ចុះឡើងអោយសព្វចូលជាតិជាមួយគ្រឿង ទុកអោយអោយពុះអោយចូលជាតិ រួចដួសចេញ ហើយជាការស្រេច ។
